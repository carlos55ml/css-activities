Respuesta pregunta: ¿Con que colores aparecenran cada una de las frases?

En la primera frase el id prevalece sobre el class y el p, asi que, al estar indicado en la hoja de estilo, se mostrara de color AZUL

Para la segunda frase, el id prevaleceria sobre el p, si este estuviera indicado en el css. Como no es el caso, se aplica el estilo de p para mostrarse de color ROJO.

En la tercera frase, domina el class, asi que se muestra de color VERDE.

En la cuarta frase, al no haber ningun parametro especial, se aplica el estilo de p para mostrarse de color ROJO.
